URDF教程 (1): URDF简介与入门

2017-11-25
ros

在ROS中，当我们要对机器人进行仿真模拟时，便需要对机器人进行建模。而URDF ( Unified Robot Description Format,意为统一机器人描述格式）正是一种用来代表机器人模型的XML代码格式。本文讲简单的对URDF进行介绍，并尝试建立一个差速轮小车的模型。教程最后会给出一系列的参考网址。在之后的教程中，我们会使用Xacro来简化URDF文件，还会添加Gazebo的相关内容，来实现Gazebo的仿真。
